# Elysia - 订阅使用指南 (GitHub Pages)

此仓库用于展示《订阅使用指南 ver.3.16》，可直接部署到 GitHub Pages。

## 部署地址 (示例)
https://bestxuyi.github.io/Elysia/

## 项目结构
```
Elysia/
├── index.html
├── assets/
│   └── banner.jpg
├── deploy.sh
├── deploy.bat
└── README.md
```

## 使用方法
1. 将项目上传到 GitHub 仓库名 `Elysia`（或修改 deploy 脚本为其他仓库名）。
2. 运行 deploy 脚本或手动 push 到 `main` 分支。
3. 在仓库 Settings → Pages 中启用 Pages（分支 main，根目录）。
