Tutorial for Building Your Own VPS Nodes

  • Common Tools: Final shell, Putty, CMD, XUI&3X-UI
  • Final shell: Buy 1 VPS (system version ubuntu22 or above) Open final shell, create a new ssh connection, enter host IP address, port, username, key to login to this VPS, then copy bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh) this link to VPS and follow prompts step by step, import generated nodes to V2rayN to use
  • Putty: Open putty, enter host IP and port to login, subsequent steps same as above
  • CMD: windows+R run CMD, then copy command ssh root@yourIP -p port (root is username, yourIP replace with your purchased host IP address, -P followed by VPS provided port) login, subsequent steps same as above
  • Recommendation: After building nodes, pair with front proxy or use traffic forwarding service to make your VPS nodes very stable and accelerated, avoid unavailability etc
