Shadowrocket:

Open Shadowrocket - Config - Top right "+" - Paste this link into URL

https://raw.githubusercontent.com/GMOogway/shadowrocket-rules/master/docs/03.shadowsocks_tiny.conf

Use this config, then on the same page open Modules - Top right "+" - Import Direct, Proxy, and Reject links respectively.

https://raw.githubusercontent.com/GMOogway/shadowrocket-rules/master/sr_direct_list.module
https://raw.githubusercontent.com/GMOogway/shadowrocket-rules/master/sr_proxy_list.module
https://raw.githubusercontent.com/GMOogway/shadowrocket-rules/master/sr_reject_list.module

After setup - Shadowrocket - Home - Global Routing - Change to "Config". Reopen Shadowrocket. Domestic apps will be Direct, Foreign apps will be Proxy (except Banking Apps).

Shadowrocket Ultimate Guide:

DOMAIN-SUFFIX,yhdm.one,PROXY

VPS Self-hosting Tutorial

  • Tools: Final Shell, Putty, CMD, XUI & 3X-UI
  • Final Shell: Buy a VPS (Ubuntu 22+), open Final Shell, create new SSH connection (IP, Port, User, Key). After login, copy the script bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh) and follow prompts. Import generated node to V2rayN.
  • Putty: Open Putty, enter Host IP and Port to login, follow steps above.
  • CMD: Win+R, run CMD, type ssh root@yourIP -p port (replace with your IP and port). Follow steps above.
  • Suggestion: Use with Front-end Proxy or Traffic Forwarding service for VPS node stability and speed acceleration.
